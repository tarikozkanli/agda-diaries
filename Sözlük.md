* Dizelge: List

* İşlev: Function
* İşlev Bileşkesi: Function composition
* serilmiş işlev: Curried function
* serilmemiş işlev: Uncurried function
* Kanıt: Proof
* Önerme: Proposition
* İç çarpım: Dot product
* Doğal: Natural number
* Doğruluk: Boolean
* Belki: Maybe
* Hangisi: Either
